Spring Boot + My Batis + Postgres Basic setup

I this article we are discussing about very minimal spring boot CRUD application integrated with postgres database using an orm framework called mybatis ( like hibernate).
Connecting our your application with a database is one of the fundamental aspects of everything you build. Especially for web applications. And it is pretty easy to build one with database connection with spring boot.
Lets look at how to setup mybatis with spring.

Step 1. Maven dependencies.

I would suggest you to go to spring application builder website and create a sample project there. Rather than adding the spring dependencies manually.
Apart form those dependencies you might need to couple of additional dependecies including mybatis-spring, mybatis and postgres driver dependency.
pom.xml 
						
<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-data-jpa</artifactId>
</dependency>
